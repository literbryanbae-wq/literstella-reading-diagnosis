# LiterStella 원서 읽기 진단 베타 v7

## 변경 요약
- PDF 저장 / 인쇄하기 버튼이 현재 페이지의 A4 리포트 영역만 인쇄하도록 수정
- 결과 카드에 원서 읽기 레벨 / 읽기 유형 / 보조 성향 설명 추가
- 추천 루틴을 텍스트 목록이 아니라 4단계 시각화 카드로 변경
- 후기 데이터 기반 동기부여 카드 3개 추가
- 리포트 디자인을 메인 컬러톤의 지적인 대시보드형으로 재정리

## 실행
```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 접속.

## 무료 자료 링크 수정 위치
`app.js`의 `FREE_RESOURCES` 안에서 `href: "#"` 값을 실제 링크로 교체.


## v10 변경 사항
- 원서 지도 탭 버튼을 대형 이모지 버튼으로 변경
- 실제 진단 결과 공유 버튼 추가(Web Share API + 클립보드 fallback)
- 하단 한글 리터스텔라 로고 영역, copyright, 카카오톡 문의 버튼 추가
- 오른쪽 클릭 및 이미지 드래그 제한 추가
- 클래식/해리포터 무료 자료 링크 연결
- 생성된 레벨별/취향별 원서 지도 이미지 포함


## v11 오늘 오픈용 정합성 수정
- Q1 목표 질문을 34~55세 여성 타깃 욕구 중심으로 교체
- L4를 `고전 정밀독해형`, L5를 `자유 독서 확장형`으로 수정
- 결과 추천 원서는 3권 원칙 유지
- 해리포터 루트는 무료 강독/무료 단어장/뉴스레터/2권 완독 클럽 구조로 유지하고 오디오북 문구 미사용
- 결과 이메일 신청을 Netlify Forms(`diagnosis-leads`)로 전송하도록 보강
- 실제 베타 오픈 CTA를 `내 원서 진단 시작하기`, `내 결과에 맞는 무료 자료 받기` 중심으로 정리

## v12 변경 사항
- 진단 결과 화면에 `나의 원서 읽기 성공 도구` 카드 추가
- 결과 유형/루트에 따라 리터스텔라 도구 사용 순서를 자동 표시
  - 해리포터 스토리 루트: 무료 강독 → 무료 단어장 → 뉴스레터 → 2권 완독 클럽 → 무료 워크북 증정 이벤트
  - 클래식 분석 루트: PDF → 강독 → 워크북 → 필사노트 → 야나완 챌린지
  - 클래식 감정/스토리/탐구 루트: 종이책/PDF, 오디오북 MP3(클래식 한정), 필사노트, 워크북, 뉴스레터, 챌린지로 분기
- PDF 리포트에도 성공 도구 카드 포함
- 이메일 리드 저장 필드에 toolkit/resources 추가
- 결과별 무료 자료 카드는 추천 루트에 맞춰 우선순위 정렬
